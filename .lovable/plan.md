

# תיקון Hero בדסקטופ — תמונה ברוחב מלא

## מצב נוכחי
- יחס 4:3 קבוע + `max-h-[600px]`
- במובייל (390px רוחב): גובה ~293px — נראה מצוין
- בדסקטופ (1920px רוחב): הגובה היה צריך להיות 1440px אבל נחתך ל-600px → התמונה נראית שטוחה ולא ממלאת את הרוחב

## פתרון
שימוש ביחס **שונה** בין מובייל לדסקטופ:
- **מובייל:** `aspect-[4/3]` — נשאר כמו שזה (293px גובה, מושלם)
- **דסקטופ:** `md:aspect-[21/9]` — יחס קולנועי שנותן ~823px גובה במסך 1920px, ו-~656px במסך 1536px. עם `max-h-[700px]` זה ייראה מרשים וממלא את הרוחב

## שינוי בקובץ `src/pages/web/WebHome.tsx`
שורה 77:
```
aspect-[4/3] md:aspect-[21/9] max-h-[700px]
```

## תוצאה
- מובייל: 4:3, גובה ~293px — תמונה גבוהה ונוחה
- דסקטופ: 21:9, גובה ~650-700px — תמונה פרוסה על כל הרוחב בלי חיתוך קיצוני
