
המשתמש מבקש שדף סיכום ההזמנה הציבורי (`/order/:orderNumber`) יציג פירוט מלא של עלות המשלוח וההנחה, לא רק את המחיר הסופי.

בדקתי את `WebOrderSummary.tsx` ו-`order-summary/index.ts`:
- ה-Edge Function כבר שולף `discount_amount` אבל **לא** שולף `shipping_cost`.
- ב-Frontend מוצגת רק שורת הנחה + סה"כ, ללא חישוב סכום ביניים (subtotal) וללא שורת משלוח.

## התוכנית

### 1. עדכון `supabase/functions/order-summary/index.ts`
להוסיף `shipping_cost` ל-SELECT של ה-orders.

### 2. עדכון `src/pages/web/WebOrderSummary.tsx` — סקציית הסיכום
לחשב סכום ביניים מהפריטים ולהציג פירוט מלא:

```
סכום ביניים (סה"כ פריטים)        ₪XXX.XX
משלוח                              ₪XX.XX   (או "חינם" אם 0)
הנחה                              -₪XX.XX   (אם קיימת)
─────────────────────────────────────────
סה"כ לתשלום                       ₪XXX.XX
```

- Subtotal = סכום `totalPrice` של כל ה-items.
- שורת משלוח תמיד מוצגת (גם אם 0 — "חינם" / "משלוח חינם" בהתאם לשפה).
- שורת הנחה מוצגת רק כש-`discount_amount > 0` (כפי שכבר קיים).
- סה"כ נשאר מודגש בתחתית.

### תרגומים
- "סכום ביניים" / "المجموع الفرعي"
- "משלוח" / "الشحن"
- "חינם" / "مجاني"

זה מספיק כדי שהלקוח יראה בדיוק איך הורכב הסכום הסופי.
